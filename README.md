# django-bookclub-app
A Django web application for managing a book club with books, authors, quizzes, and a dynamic member ranking system.

# Description
A Django-based web application for managing a book club. The platform presents monthly book and author selections, allows members to participate in quizzes related to the books, and displays rankings based on quiz performance.

# Technologies
- Python
- Django
- PostgreSQL
- Bootstrap
