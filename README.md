Book Club Django Application

Description
This project is a Django web app built as the final assignment for a Django Basics course.
It serves as a simple book club platform where users can look through a list of books, manage club members, take quizzes, and see how members rank based on their quiz results.

Technologies Used
- Python 3
- Django
- PostgreSQL
- HTML (Django Template Engine)

Application Structure
The project is organized into three Django apps:
- books – handles books and authors
- members – manages club members and ranking logic
- quizzes – provides quiz functionality and score submissions

Features
- Full CRUD operations for books and members
- Support for many‑to‑one and many‑to‑many relationships
- Custom forms with validation, help text, and disabled fields
- Quiz score submission with automatic ranking updates
- Custom Django template filter
- Custom 404 page
- Base template with shared navigation and footer
- PostgreSQL integration
- All pages accessible through the main navigation
